MyBatis学习日记(第一天)

2025/1/2 Java开发学习

开发我的第一个 MyBatis 程序

1. resources 目录

  • 放在这个目录当中的一般都是资源、配置文件。
  • 直接放在 resources 目录下的资源等同于放到类的根路径下。

2. 开发步骤

  1. 打包方式：jar

  2. 引入 MyBatis 依赖，MySQL 驱动。

  3. 编写 MyBatis 核心配置文件：mybatis-config.xml

    • 这个文件名可以被替换，只是约定俗成的这个名字。
    • 文件的存放位置也不是固定的，但一般情况下被放到类的根目录。
    • 不理解 XML 文件中的信息也没关系，先修改连接数据库的配置信息即可。

  4. 编写 XxxxMapper.xml

    • 在这个文件中编写 SQL 语句。

    • 文件名和存放位置也不是固定的。

  5. 在 mybatis-config.xml 文件中指定 XxxxMapper.xml 文件的路径：

    <mapper resource="CarMapper.xml"/>
    • 注意：resource 属性会自动从类的根路径下开始查找。

  6. 编写 MyBatis 程序（使用 MyBatis 类库编写 MyBatis 程序，连接数据库，做增删改查）。

3. 从 XML 中构建 SqlSessionFactory

通过官方文档的这句话你能想到什么呢？

  1. 在 MyBatis 中一定有一个很重要的对象，这个对象是 SqlSessionFactory 对象。
  2. SqlSessionFactory 对象的创建一定需要 XML，XML 是一个配置文件。

通过 mybatis-config.xml 创建 SqlSessionFactory 对象。MyBatis 中有两个主要的配置文件：

  1. mybatis-config.xml：核心配置文件，主要配置数据库的连接等（只有一个）。
  2. XxxxMapper.xml：专门用来编写 SQL 语句的配置文件（一个表有一个），例如：
    • 一个 User 表对应一个 UserMapper.xml，
    • 一个 Student 表对应一个 StudentMapper.xml。
